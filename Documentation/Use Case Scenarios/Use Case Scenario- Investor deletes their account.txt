Use Case Scenario
Investor deletes their account

Context: Investor is able to delete their account if they so desire. 

Primary Flow: 
1. Investor selects delete account option. 
2. System detects delete account requests. 
3. System displays dialog box asking for confirmation of deletion. 
4. System checks the investors portfolio if they have any holdings. 
5. System detects no holdings. 
6. System removes the account and redirects user to the homepage as user. 

Alternative Flow:
I. System detects one or more holdings in investor's portfolio. 
A6. System displays the information and asks investor to sell before deleting account. 
A7. Returns to the investor's portfolio.
